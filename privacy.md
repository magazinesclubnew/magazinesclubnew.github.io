    <h2>我们收集的信息</h2>
    <p>本站使用 Microsoft Clarity 分析工具来了解访客如何使用网站。当您同意使用 Cookie 时，我们可能会收集以下信息：</p>
    <ul>
      <li>页面浏览量和访问时长</li>
      <li>点击和滚动行为</li>
      <li>设备类型和浏览器信息</li>
      <li>大致地理位置（国家/地区级别）</li>
    </ul>

    <h2>我们如何使用这些信息</h2>
    <p>收集的数据仅用于：</p>
    <ul>
      <li>改善网站内容和用户体验</li>
      <li>了解哪些文章最受欢迎</li>
      <li>识别并修复网站问题</li>
    </ul>

    <h2>Cookie 和追踪</h2>
    <p>本站使用 Microsoft Clarity 进行网站分析。只有在您明确同意后，我们才会加载分析脚本。您可以随时通过清除浏览器 Cookie 或使用隐私模式来撤回同意。</p>

    <h2>第三方服务</h2>
    <p>本站使用以下第三方服务：</p>
    <ul>
      <li><strong>Microsoft Clarity</strong>：网站分析和热力图（<a href="https://privacy.microsoft.com/privacystatement" target="_blank">Microsoft 隐私声明</a>）</li>
      <li><strong>GitHub Pages</strong>：网站托管</li>
      <li><strong>Google Fonts</strong>：字体服务</li>
    </ul>

    <h2>您的权利</h2>
    <p>根据 GDPR，您有权：</p>
    <ul>
      <li>拒绝 Cookie 和追踪</li>
      <li>请求访问我们收集的关于您的数据</li>
      <li>请求删除您的数据</li>
    </ul>
